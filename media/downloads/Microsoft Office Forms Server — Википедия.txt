Forms Server 2007


Microsoft Office Forms Server — серверный продукт компании Microsoft, позволяющий просматривать и заполнять формы, разработанные в приложении Microsoft InfoPath[источник не указан 1016 дней], через интернет с помощью браузера[1], включая браузеры для мобильных устройств. Формы в Microsoft Forms Server 2007 поддерживают проверку вводимых данных (валидацию данных) и условное форматирование, но допустимы не все возможности форм InfoPath. Microsoft Forms Server позволяет лишь просматривать, заполнять формы и отправлять заполненные данные на сервер, но не создавать новые шаблоны форм — для этого по-прежнему используется InfoPath.
Microsoft Office Forms Server 2007, по сути, является надстройкой над Windows SharePoint Services и требует для своей работы .NET Framework 2.0, Internet Information Server, Windows SharePoint Services 3.0 и Microsoft SQL Server. Альтернативой Forms Server является Microsoft Office SharePoint, в редакцию Enterprise которого также входит функциональность по работе с формами InfoPath.
